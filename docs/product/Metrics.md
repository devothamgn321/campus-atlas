# CampusOS Metrics Framework

## North Star Metric

Weekly active students completing meaningful campus workflows.

## Activation Metrics

- New student onboarding completion rate
- Profile completion rate
- First workflow completion rate
- First AI copilot query completion

## Engagement Metrics

- Weekly active students
- Monthly active students
- Workflows completed per student
- AI assistant sessions per user
- Service module usage frequency

## Operational Metrics

- Manual administrative request reduction
- Average workflow completion time
- Document verification turnaround time
- Admin dashboard weekly active users
- Department-level service resolution time

## Satisfaction Metrics

- Student satisfaction score
- Admin satisfaction score
- AI answer helpfulness rating
- Repeat usage rate

## Business / Institutional Metrics

- Cost saved through workflow automation
- Service adoption across departments
- Number of integrated campus services
- Reduction in duplicate systems
