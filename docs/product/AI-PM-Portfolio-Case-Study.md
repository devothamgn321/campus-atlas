# AI Product Manager Portfolio Case Study

## Role

AI Product Manager for CampusOS, an AI-powered digital campus operating system for higher education.

## Problem

Students interact with fragmented university systems: registrar portals, library tools, dining systems, transport desks, career portals, and manual help desks. This creates avoidable support load, slow verification, poor student experience, and weak operational visibility.

## Product Decision

The highest-leverage AI wedge is not a generic assistant. It is a campus-specific copilot connected to workflows, identity, verified data, and admin intelligence.

## What This Project Demonstrates

- AI use-case prioritization
- RAG system design
- AI evaluation planning
- Responsible AI governance
- Experiment design and decision rules
- KPI design and dashboard thinking
